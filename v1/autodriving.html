In this project, we follow "Perception -- Planning -- Control" technical roadmap. To ensure the practicality, stability and safety, we put no limitation on the type of mature tools and sensors. We use Raspberry Pi Model 3B as Central Control Module.

I list here a few features to be implemented recently.

  • Assemly of car's machenical structure, including wheels, motors, chassis and necessary soldering.
  • Basic circuits and programming of motor control, using L298N DC model control.
  • Assembly of GPS module and GPS signal receival.
  • Real-time routing, GPS tracking and visualization.
